**Проблема:** Redis без пароля, порт открыт на 0.0.0.0:6379

**Исправление:**
1. Сгенерировать пароль
2. Добавить `--requirepass` в command redis
3. Обновить `REDIS_URL`
4. Закрыть порт: `"127.0.0.1:6379:6379"`
5. Перезапустить redis

**Статус:** 🔴 Не сделано

### 1.2. PostgreSQL — сменить пароль и закрыть порт

**Проблема:** Пароль `postgres`, порт открыт на 0.0.0.0:5432

**Исправление:**
1. Сгенерировать новый пароль
2. `ALTER USER postgres WITH PASSWORD ...`
3. Обновить .env и DATABASE_URL
4. Закрыть порт: `"127.0.0.1:5432:5432"`

**Статус:** 🔴 Не сделано
